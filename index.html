Learning Git and Github

We are going to learn the basics of Git and Github

Basic concepts

  • Repository: Source code of the proyect. Contains all the files that are going to be tracked.
  • Clone repo: Create a copy of an existing Git repository.
  • Commit: Snapshots of the code.
  • Branch: Represents an idependent line of development.
  • Pull Request: Event in git where a contributor requests a review of the changes done.
